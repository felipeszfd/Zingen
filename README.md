# Zingen - Karaokê

Este projeto foi desenvolvido seguindo um guia da Rocketseat e tem como objetivo apresentar uma landing page moderna para o aplicativo Zingen, um app de karaokê.

## Sobre o Projeto
O Zingen é uma aplicação fictícia de karaokê, com foco em design responsivo, boas práticas de HTML e CSS, e experiência do usuário. O projeto demonstra organização de estilos, uso de variáveis CSS, grid, flexbox e componentes reutilizáveis.

## Estrutura
- `index.html`: Página principal do projeto
- `assets/`: Imagens, ícones e outros recursos visuais
- `styles/`: Folhas de estilo CSS organizadas por seções e componentes

## Principais Seções
- Hero: Apresentação do app
- Sobre: Informações sobre o aplicativo
- Funcionalidades: Cards com recursos do app
- Planos e preços: Tabela de preços
- Download: Chamada para baixar o app
- Footer: Rodapé com links e redes sociais

## Tecnologias Utilizadas
- HTML5
- CSS3
- Fonte: [Inter](https://fonts.google.com/specimen/Inter)

## Como executar
Basta abrir o arquivo `index.html` em seu navegador preferido. Não há dependências externas além das fontes do Google Fonts.

## Créditos
Projeto criado seguindo o guia da Rocketseat para estudos de HTML e CSS.
